Rsync

If you’re looking for the most effective means to copy files from one server to another, then look no further than rsync. Rsync is a Linux utility that enables you to copy files locally or even between servers. Because rsync uses SSH to connect it’s more secure than standard FTP (as secure as SFTP) and much faster. Some hosts require SSH keys to connect, but most simply need the hosting account username and password. This is particularly true with hosting providers that use cPanel. Another benefit to rsync is that it makes quick work of amending any discrepancies between the two directories targeted in the sync, copying only files that need it. Rerunning the command is virtually harmless, ensures both directories match exactly, and resolves quickly if there’s little-to-no work to do.

For more detail and to learn about additional flags you can use with the rsync command, check the man page.

MySQL, MySQLDump, and WP CLI

You can perform an entire WordPress migration from your terminal. The mysql utility allows you to, not only, execute MySQL commands from your command prompt, but to also import an entire database with one command. Likewise, the mysqldump utility lets a user export an entire database, or specific terminals, with the proper syntax. WP CLI has support for both these functions using the db import and db export commands and doesn’t require passing arguments for the database name, user, or password since it will read those values from the wp-config.php file.

For more information please check out the WP CLI documentation.
